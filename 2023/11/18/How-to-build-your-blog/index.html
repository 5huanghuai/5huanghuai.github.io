How to build your blog?

环境依赖

  1. git
  2. hexo
  3. nodejs
  4. ssh

环境搭建

创建本地博客

主题更换

从github 下载想要的主题

更改‘~/blog_name/_config.yml’文件中的themes的内容

本地渲染

创建必要的页面

更改~/blog_name/source/about/index.md文件内容

gitub端部署

ssh连接github

  1. 生成ssh密钥
  2. github设置
    github -> settings -> SSH and GPG keys -> New SSH key
    将复制的内容粘贴至key目标框中，Title可以为任意内容。
  3. 验证 当终端输出不报错，则表示使用ssh成功连接至github

github创建仓库

github -> your respositories -> New
repositry name 必须为your_github_name.github.io

将本地博客部署到github

  1. 再次更改~/bolg_name/_config.yml文件
  2. 渲染本地博客并同步到github
  3. 部署成功后，网址
    http://5huanghuai.github.io

添加新文章

  1. hexo new ‘文章名称’
  2. 在～/blog_name/source/_posts/文章名称.md 修改文章内容
